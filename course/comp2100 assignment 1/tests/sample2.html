This research investigates the role of interface manipulation style on reflective cognition and concept learning through comparison of the effectiveness of three versions of a software application for learning two-dimensional transformation geometry. The three versions, respectively, utilize a Direct Object Manipulation (DOM) interface - in which the user manipulates the visual representation of objects being transformed; a Direct Concept Manipulation (DCM) interface - in which the user manipulates the visual representation of the transfroamtion being applied to the object; and a Reflective Direct Concept Manipulation (RDCM) interface - in which the DCM approach is extended with scaffolding.
